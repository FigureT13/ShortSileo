# ShortSileo
A jailbreak package mananager based off of a shortcut based jailbreak tool

# Build your own repo

Repo Scheme:
`{
  "Name": "My Awesome Repo",
  "Packages": {
    "AdBlocker": {
      "url": "https://example.com/adblock",
      "size": "5.4 MB",
      "minIOS": "14.0",
      "maxIOS": "17.0",
      "description": "Block all ads system-wide."
    }
  }
}`
